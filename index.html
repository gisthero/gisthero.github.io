Features

Features

Gist Management

Manage, Create, Delete, and Edit Your Gists with Ease

Use GistHero to manage, create, delete, and edit gists on your iPad. Add extra files to your gist, edit them, and save them. It's that simple. Discover the gists of the people that you follow and your followers. Star their gists or fork them. Be social!

Heroku Deployment

Deploy Your Gists to Heroku Directly from Your Device

Use GistHero to deploy your gists as web sites or web services to Heroku. Directly and securely from your iPad. Deploy to an existing Heroku app or a new one. And take a look at the deployment log. Be shipping!

Code Editor

Write Code Using the Editor with Syntax Highlighting

Use the GistHero code editor to write your next micro web sites or web services. It's fast, responsive, and it can edit large files, say for example, a 10000-line source file. The code editor features syntax highlighting for Python, Ruby, HTML, CSS, JavaScript, C, Java, Objective-C, PHP and more. Be creative!
